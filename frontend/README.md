# Frontend

React + TypeScript dashboard for the one-camera anti-cheat demo.

## Responsibilities

- access the laptop webcam from the browser
- start and stop a monitoring session
- send sampled frames to the FastAPI backend
- render the raw preview, annotated preview, and monitoring logs

## Run

```bash
cd /Users/huy.dao/XuLyAnh/anti-cheat-demo/frontend
cp .env.example .env
npm install
npm run dev
```
